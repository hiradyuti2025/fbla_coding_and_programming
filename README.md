# Hometown Hub

**A beautiful, interactive local business directory** focused on supporting small businesses, Black-owned, and minority-owned establishments.

## Features

- **Interactive Map** with custom pins and popups
- **Directory View** with filtering and sorting
- **Saved Businesses** (favorites)
- **Business Detail Modals** with reviews, deals, and ratings
- **AI Chat Assistant** (Claude-powered with API key support)
- **Dark Mode** support
- **Fully Responsive** design
- **Community-focused** design celebrating local businesses

## Design Updates

- **Typography**: Playfair Display for headings, DM Sans for body text
- **Category Colors**: 18 distinct, carefully chosen colors for better visual distinction
- **Clean UI**: Removed redundant "Sort By" sections and generic text

## Categories

Includes 18 business categories with unique colors:
- Food & Dining, Retail, Services, Health, Fitness, Beauty, Automotive, Education, Entertainment, Pets, Home & Garden, Electronics, Apparel, Grocery, Nightlife, Professional Services, Black-Owned, Minority-Owned.
